# Analyse des indicateurs de l’égalité femmes/hommes

## Contexte et objectifs

Ce projet vise à collecter, nettoyer et analyser les données RH afin de calculer les indicateurs d’égalité professionnelle tout en respectant les principes du RGPD.

## Soft Skills
- Respect strict de la confidentialité  
- Communication claire sur des indicateurs sensibles  
- Rigueur dans la validation des données  
- Structuration méthodique de l’analyse  

## Compétences
- Nettoyage et standardisation des données  
- Définition de règles de gestion  
- Calcul d’indicateurs RH normés  
- Vérification de la cohérence et de la fiabilité des données  

## Outils utilisés
- Python ou Excel  
- Jupyter Notebook
